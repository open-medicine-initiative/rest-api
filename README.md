#Requirements

* iojs >= 1.8.1 (tip: use latest nvm to manage environments)
* phantomjs

#Installation

`sudo npm install -g gulpjs/gulp-cli#4.0 babel coffee-script pm2`
`npm install`

#Run

Single server: 'node startup.js'
Clustered : `pm2 start server.js -i 0 --next-gen-js`
